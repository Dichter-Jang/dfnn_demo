# dfnn_demo
demos for multimodal fusion program
1. demo1: demo1_gpjm_generate_synthetic_data\demo1_gpjm_generate_synthetic_data.ipynb

Overview of how I using Gaussian process joint modeling framework to generate the synthetic dataset.

2. demo2: demo2_synthetic_data_train_and_eval\demo2_synthetic_data_train_and_eval.ipynb

Overview of using fusion model Bimodal Deep AutoEncoder to train the synthtic dataset, and using SVM/PCA/TSNE to evaluate the results.

3. demo3: demo3_bbbd_data_train_and_eval\demo3_bbbd_data_train_and_eval.ipynb

Overview of using fusion model Bimodal Deep AutoEncoder to train the real dataset, using SVM/PCA/TSNE to evaluate the results, and do the bootstrap to estimate perforrmance stabiliy.
